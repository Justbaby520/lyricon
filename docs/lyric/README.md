# Lyric 文档

Lyric 文档覆盖 Lyricon 的歌词数据模型、Bridge 接入和显示相关能力。

## Bridge

Bridge 提供两类接入能力：

- **Provider**：音乐播放器或歌词插件向 Lyricon 推送歌曲、歌词、播放状态和显示配置。
- **Subscriber**：第三方应用订阅 Lyricon 当前活跃播放器、歌曲、歌词和播放状态。
